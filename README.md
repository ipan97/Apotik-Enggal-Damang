# SpringBoot with AdminLTE 2
---

Aplicação com SpringBoot e o template AdminLTE 2 para facilitar o desenvolvimento.


Características
---

* Spring Boot;
* Spring Security para login básico com permissões;
* Thymeleaf para view;
* Banco Mysql ou outros;
* Crud básico de clientes;
